# Climate Heterogeneity, Environmental Conditions, and Economic Development Across Brazilian Microregions

This project uses an observational design to examine multivariate differences among Brazilian microregions in 2019. The analysis focuses on environmental, climatic, and socioeconomic factors originally compiled for dengue-related research. The goal is to describe how Brazilian microregions
differ in their combined climate, environmental, and built-environment profiles, and to determine whether these differences can be summarized, grouped, and formally tested using multivariate statistical methods.

The data used in this study were obtained from the Zenodo dataset “[Spatiotemporal dataset of dengue influencing factors in Brazil based on geospatial big data cloud computing](https://doi.org/10.5281/zenodo.15091652)”.
